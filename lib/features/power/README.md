# power

For controlling the power on/off state of the DUT.

## Implementations

- `autokitRelay`
- `crelay`
- `jetsonTx2` (specifically for use with jetson Tx2 dev kit)
- `jetsonOrinNxEs` (specifically for use with use with this device type)
- `JetsonOrinNxXv3` (specifically for use with this device type)
- `dummyPower`: for use when no device is connected
